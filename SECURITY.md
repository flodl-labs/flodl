# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in floDl, please report it responsibly
by using GitHub's private advisory mechanism rather than opening a public issue.

Contact: Open a [private security advisory](https://github.com/flodl-labs/flodl/security/advisories/new) on GitHub.

We will acknowledge receipt within 48 hours and aim to provide a fix or
mitigation within a reasonable timeframe depending on severity.
