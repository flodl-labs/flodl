Reproduce

Terminal
$ git clone https://github.com/fab2s/fbrl.git
$ cd fbrl && git checkout 102225b

# PyTorch
$ cd python && make generate && make train CONFIG=runs/letters/v7-void-repulsion/config.yaml CKPT=50

# flodl
$ cd .. && make train-letter DATA=../python/data/letters EPOCHS=100 SAVE=training MONITOR=3000

All raw artifacts (per-epoch timings, logs, pseudocode, models) are committed at fbrl@102225b. See the full benchmark report for methodology and honest accounting of asymmetries.
