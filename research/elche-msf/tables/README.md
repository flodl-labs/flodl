# Eval-focused paper tables

Curated tables for the manuscript, each derived from upstream sweep
aggregators. Numbers are quoted from the canonical `aggregate.txt` /
`per_cell.csv` of each sweep — re-running those aggregators / extract
scripts regenerates the underlying numbers; this directory is the
human-readable rollup.

## Figures

Each table embeds the most informative figure(s) from its source
sweep's `data/<sweep>/analysis/` dir. Every sweep now has a per-sweep
`analyze.py` that produces per_cell.csv, per_rank.csv, and the
embedded figure(s); the cross-sweep Pearson scatter for the
framing-validity table is generated by `tables/fig_framing_gates.py`.

To regenerate everything image-side from the current extracts:

```
python3 research/elche-msf/data/passive-observation/analyze.py
python3 research/elche-msf/data/relaxed-anchor/analyze.py
python3 research/elche-msf/data/cliff-bracket/analyze.py
python3 research/elche-msf/data/pareto-frontier/analyze.py
python3 research/elche-msf/data/cpu-async-multiseed/analyze.py
python3 research/elche-msf/data/resnet56-cpu-async/analyze.py
python3 research/elche-msf/data/cpu-async-alpha-sweep/analyze.py
python3 research/elche-msf/tables/fig_framing_gates.py
python3 research/elche-msf/tables/cross_sweep.py
```
